Presentation Layer – Making the Web Look Good

The presentation layer is what users see and use on a website. It includes everything visible like layout, colors, text, images, buttons, and animations. This is also called the frontend of a website.

1. HTML – The Skeleton

HTML stands for HyperText Markup Language. It gives structure to the webpage. Think of it like the skeleton or base of a website.

Example: A heading, a paragraph, an image, a list – all are created using HTML.

2. CSS – The Style

CSS stands for Cascading Style Sheets. It is used to style HTML. It adds colors, spacing, fonts, borders, and layout designs to the page.

Think of CSS as decorating your website and making it look attractive.

3. JavaScript – The Interaction

JavaScript is used to make a website interactive. It helps when you want buttons to do something, forms to work, popups to appear, or anything that changes on user action.

It makes the webpage dynamic and alive.

4. React – Smart Way to Build UI

React is a JavaScript library made by Facebook. It is used to build user interfaces. It helps you create small parts called components like buttons, cards, and menus, which you can reuse easily.

React is fast, efficient, and widely used for building modern web applications.

5. Bootstrap – Quick and Responsive Design

Bootstrap is a CSS framework. It gives ready-made design blocks like buttons, forms, grids, and layouts. It helps build websites quickly and make them look good on both mobile and desktop screens.

It saves time and makes your website responsive without writing much CSS from scratch.

Summary

HTML gives structure.

CSS adds style.

JavaScript adds interaction.

React builds reusable UI components.

Bootstrap offers ready-made design blocks.

In short:

HTML = Structure

CSS = Style

JavaScript = Behavior

React = Smart Components

Bootstrap = Fast Design
